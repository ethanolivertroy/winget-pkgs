# WinGet Package Repository

This repository contains Windows Package Manager (winget) manifest files for ethanolivertroy's applications.

## What is WinGet?

WinGet is Microsoft's official package manager for Windows. It allows users to discover, install, upgrade, remove, and configure applications through a command-line interface.

## Available Packages

- **kevs-tui**: A terminal user interface application

## Installation

To install a package from this repository using winget:

```powershell
winget install ethanolivertroy.kevs-tui
```

## Repository Structure

Manifest files are organized by publisher and package name:
